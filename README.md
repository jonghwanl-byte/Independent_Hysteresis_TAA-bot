# Independent-Hysteresis-TAA (Sharpe 1.75)

2004년부터 2025년까지 20년 이상의 데이터를 기반으로 검증된, **독립적이고(Independent) 견고한(Robust)** 전술적 자산 배분(TAA) 전략입니다.

이 전략은 복잡한 거시경제 지표(금리 등)를 사용하지 않고, **각 자산(`QQQ`, `TLT`, `GLD`)이 자신의 가격 추세(이동평균선)만을 보고 독립적으로 투자 비중을 조절**합니다. 여기에 `+/- 3%` 이격도 밴드를 적용하여 횡보장에서의 잦은 매매(휩쏘)를 방지합니다.

## ⚙️ 전략 핵심 룰 (Strategy Rules)

### 1. 자산 배분 (Base Weights)
이 전략은 다음 3가지 자산에 고정된 기본 비중을 할당합니다.

* **`QQQ` (성장주): 45%**
* **`TLT` (장기채): 35%**
* **`GLD` (금): 20%**

### 2. 독립적 MA 신호 (Independent MA Signals)
각 자산은 **자신의 가격**과 **자신의 이동평균선(MA)**만을 비교하여 투자 여부를 결정합니다. (다른 자산이나 금리의 영향을 받지 않습니다.)

* **MA 지표:** `20일`, `120일`, `200일` 이동평균선

### 3. 이격도 밴드 룰 (Hysteresis Band)
횡보장에서의 잦은 매매(Whipsaw)를 방지하기 위해 **`+/- 3%`**의 "데드 존(Dead Zone)"을 설정합니다.

* **ON (매수):** 가격이 MA선보다 **+3% 위로** 올라갈 때
* **OFF (매도):** 가격이 MA선보다 **-3% 아래로** 내려갈 때
* **HOLD (유지):** 가격이 **MA선 `+/- 3%` 사이**에 있을 때는 **기존 신호 유지**

### 4. 비중 조절 (Weight Scaling)
각 자산별로 3개의 MA 신호(ON/OFF) 개수에 따라 실제 투자 비중을 결정합니다.

| ON 신호 개수 | 투자 비중 (Scaling) | 예시 (QQQ 45% 기준) |
| :---: | :---: | :---: |
| **3개** | **100%** | 45.00% 투자 |
| **2개** | **75%** | 33.75% 투자 |
| **1개** | **50%** | 22.50% 투자 |
| **0개** | **0%** | 0.00% (현금화) |
